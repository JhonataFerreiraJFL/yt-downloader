# 📥 YouTube Downloader (yt-dlp wrapper)

A simple Python tool to download YouTube videos in a user-friendly way, using yt-dlp under the hood.

---

## 🚀 Features

- Download videos from a URL
- Choose maximum quality (e.g., 1080p, 720p)
- Automatically save files to a selected folder
- Simple command-line interface
- Compatible with Linux, Windows, and macOS

---

## 📦 Requirements

- Python 3.8+
- yt-dlp

Installation:

```bash
pip install -r requirements.txt
